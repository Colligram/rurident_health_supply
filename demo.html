🚀 Features Included

  • Scroll-triggered GSAP animations with ScrollTrigger
  • Smooth scroll scrubbing (animations reverse when scrolling up)
  • Responsive design for desktop and mobile
  • Layered storytelling with dental chair, patient, dentist, and tools
  • Performance optimizations for mobile devices
  • Clean, modular code structure
  • Debug mode for development
  • Preloading for optimal performance

🎬 Animation Sequence

As users scroll, the story unfolds:

  1. Background fades in - Clinical environment
  2. Dental chair slides in from left - Modern equipment
  3. Patient appears - Comfortable and relaxed
  4. Dentist enters from right - Professional care
  5. Tools float in one by one - Precision instruments
  6. Text overlays appear - Key messages
